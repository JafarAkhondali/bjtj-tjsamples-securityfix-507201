# Ubuntu Installation #

* https://caffe.berkeleyvision.org/install_apt.html


## For Ubuntu (>=17.04) ##

```
sudo apt install caffe-cpu
sudo apt install caffe-cuda
```

**For development (e.g. Header files)**

```
sudo apt install libcaffe-cpu-dev
sudo apt install libcaffe-cuda-dev
```
