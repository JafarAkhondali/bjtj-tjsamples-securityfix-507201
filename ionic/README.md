# ionic framework #

<https://ionicframework.com/>

## install ##

``` shell
npm i -g @ionic/cli
```

## create app ##

``` shell
ionic start
```

## start app ##

``` shell
$ cd myApp
$ ionic serve
```

## deploy to android ##

``` shell
npm install @capacitor/android

npx cap add android

npm run build
```


### run ###


``` shell
npx cap run android
```
